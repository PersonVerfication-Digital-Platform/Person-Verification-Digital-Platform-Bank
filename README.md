<h2>Developer Guide</h2>
<p>
<h3>1. Prerequisites:</h3>
<ul>
<li>NodeJS</li>
<li>NPM</li>
</ul>
</p>


<h1></h1>

<p>
<h3>2. Install Packages:</h3>

  + Installing packages in front end

    `$ cd client`

    `$ npm install`

  + Installing packages in back end


    `$ cd server`

    `$ npm install`

</p>

<h1></h1>


<p>
<h3>3. NPM Commands:</h3>

+ Start Front end          

  `$ cd client`

  `$ npm start`

+ Start Back end   

  `$ cd server`

  `$ npm start`


</p>
